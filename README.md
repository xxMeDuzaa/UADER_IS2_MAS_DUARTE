# Gestión de la Configuración y Programación Python
Este repositorio ha sido creado para la asignatura **Ingeniería de Software II** de la Facultad de Ciencia y Tecnología (FCyT - UADER). El propósito principal es gestionar las versiones de los scripts desarrollados en Python y documentar el proceso de configuración del entorno de desarrollo.

## 1. Requisitos del Entorno
Para ejecutar los proyectos contenidos en este repositorio, se requiere la instalación de los siguientes paquetes de software:

1. Python 3 (descargado desde python.org).
2. Pip3 para la gestión de paquetes.
3. Git para el control de versiones.
4. Matplotlib para la generación de gráficos.

### 1.1 Pasos de Instalación y Configuración
A continuación se detallan las acciones realizadas para configurar el espacio de trabajo local:

**Pasos iniciales de configuración:**
1. Instalación de intérprete Python y herramientas de Git.
2. Clonación del repositorio mediante la terminal Git Bash.
3. Creación de la estructura de carpetas (`src`, `doc`, `bin`).

**Tareas de sincronización con GitHub:**
1. Ejecución y prueba local de los scripts `primos.py` y `factorial.py`.
2. Uso de comandos `git add` y `git commit` para registrar cambios.
3. Recuperación de archivos tras simulacros de borrado accidental.
